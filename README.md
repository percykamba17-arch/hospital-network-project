# Hospital Network Project

Design and Simulation of a Hospital Network Using Cisco Packet Tracer.

## Overview
This project presents the design and simulation of a multi-floor hospital network built in Cisco Packet Tracer. The network is structured to support multiple hospital departments through VLAN segmentation, inter-VLAN routing, dynamic routing, DHCP services, and secure remote access.

## Objective
The objective of this project was to design a scalable, organized, and secure hospital network environment that allows efficient communication between departments while maintaining proper network segmentation and management.

## Network Structure
The simulated hospital network is distributed across three floors:

### Floor 3
- Administration
- Doctors

### Floor 2
- Emergency
- Pharmacy

### Floor 1
- Radiology
- Patients WiFi
- Server Network

## Technologies Used
- Cisco Packet Tracer
- VLANs
- Inter-VLAN Routing (Router-on-a-Stick)
- OSPF
- DHCP
- SSH
- Serial WAN Links

## Skills Demonstrated
- Network design and segmentation
- IP addressing and subnetting
- Switch and router configuration
- Dynamic routing with OSPF
- DHCP configuration
- Secure remote management with SSH
- Network testing and connectivity verification
